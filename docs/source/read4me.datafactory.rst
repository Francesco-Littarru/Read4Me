Create a dataset - `datafactory` module
=======================================
.. _datafactory_mod:
.. note::
    You can create a corpus and its Tfidf model using the script read4me/scripts/make_corpus.py.
    From the activated environment, call the script with the -h flag to see the help page:

    .. code-block:: console

        (venv) user@userpc:~$ python -m read4me.scripts.make_corpus -h


    The parameters used by the script are coherent with the parameters used in this module, see below for more information.
